# T2 Narrow Placement

## Purpose
Measure lateral balance and foot placement precision in a constrained stepping corridor.

## Assembly Steps
- Build the flat lane as in T1.
- Mark the 220 mm stepping corridor relative to lane centerline with tolerance under 2 mm.
- Verify the start and finish boxes are centered on the corridor.

## Surface Guidance
- Surface: sealed birch plywood with matte lane markings
- Target friction coefficient: 0.60-0.75

## Start / Finish
- Start pose: Robot stands centered in the corridor with feet parallel and inside the marked start box.
- Finish condition: Robot reaches the end box while all stance contacts remain inside the stepping corridor.
