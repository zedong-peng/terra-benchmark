# T7 Repeated Obstacles

## Purpose
Measure repeated swing-foot clearance and short-horizon adaptation across several low obstacles.

## Assembly Steps
- Assemble the flat lane as in T1.
- Install placement guides so each obstacle can be repositioned repeatably.
- Place five bars at the prescribed spacing measured from leading edge to leading edge.

## Surface Guidance
- Surface: flat plywood lane with removable HDPE or wood obstacle bars
- Target friction coefficient: 0.60-0.75

## Start / Finish
- Start pose: Robot starts 400 mm before the first obstacle with stance centered on the lane.
- Finish condition: Robot traverses the full obstacle series and crosses the finish line without fall.
